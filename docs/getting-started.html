Getting Started

Prerequisites

  • Go 1.21+
  • ffmpeg with drawtext support: brew install ffmpeg-full
  • API keys (set in .env in your project directory or ~/.puffinship/.env):
Variable Required for
PUFFINSHIP_ANTHROPIC_API_KEY AI cut, transcript correction, broll analysis
PUFFINSHIP_OPENAI_API_KEY Whisper transcription
PUFFINSHIP_GOOGLE_API_KEY Veo 3 broll generation (optional)
PUFFINSHIP_HEYGEN_API_KEY HeyGen avatar speaker (optional)
PUFFINSHIP_ELEVENLABS_API_KEY ElevenLabs TTS (optional)

Install

git clone <repo>
cd puffinship
go build -o puffinship ./cmd/puffinship

Project layout

Create a directory for your project. Puffin Ship expects this structure:

my-project/
├── puffinship.yaml          # your config
├── speaker.MP4          # source footage
├── broll/               # existing broll clips (optional)
├── music.mp3            # background music (optional)
└── .puffinship/             # created automatically — cache and generated files

Your first pipeline

This minimal pipeline transcribes your footage, makes an AI content cut, and exports to Final Cut Pro.

Create my-project/puffinship.yaml:

version: 1

project:
  name: "My First Edit"
  resolution: [1080, 1920]   # portrait — swap to [1920, 1080] for landscape
  fps: 30
  sample_rate: 48000

input:
  - id: speaker-footage
    select: "speaker.MP4"

timeline:
  levels:
    - name: main
      order: 1
      type: any
      backbone: true

pipeline:
  # Step 1: transcribe the footage
  - id: transcribe
    processor: generate_transcript
    input:
      - id: source
        file: speaker-footage
    params:
      language: "en"
    cache: true

  # Step 2: AI content cut — Claude reads the transcript and removes filler/dead air
  - id: ai_cut
    processor: ai_agent
    input:
      - id: transcript
        step: transcribe
      - id: speaker
        file: speaker-footage
    prompt: |
      You are an editor making a content cut. The transcript and silence intervals are
      already provided in the context block above. Call get_input("speaker") to get the
      source filename, then call create_edit_decision for each segment to keep.
      Use source=files[0], type="video", level="main". Snap in/out to silence boundaries.
      Remove: filler words, long pauses, false starts. Keep: complete sentences.
    output:
      timeline: main
    review: true
    cache: true

output:
  - save_finalcut:
      path: "./output/project.fcpxmld"

Run it

cd my-project
../puffinship run

Puffin Ship runs each step in order. The ai_cut step has review: true — it will pause and show you the edit decisions:

[a]pprove  [r]eject  [e]dit in vi  [s]kip

Press a to approve and continue. When the run finishes, open output/project.fcpxmld in Final Cut Pro.

Other commands

./puffinship validate                   # validate config without running
./puffinship inspect --step ai_cut      # ASCII timeline visualization
./puffinship cache clear                # clear all cached step outputs
./puffinship cache clear --step ai_cut  # clear one step
./puffinship -c other-project.yaml run  # explicit config path

Web app

Puffin Ship also runs as a web server with a React UI:

./puffinship serve
