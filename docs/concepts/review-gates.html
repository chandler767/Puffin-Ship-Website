Review Gates

What is a review gate?

Any step can be paused for human review before the pipeline continues. Add review: true to the step:

- id: ai_cut
  processor: ai_agent
  ...
  review: true

After the step executes, Puffin Ship shows the output and waits for your decision.

Web app review

In the web app, a step with review: true pauses the run:

  1. The step output is serialized to JSON
  2. A run_reviews row is created with status='pending'
  3. The run status becomes awaiting_review
  4. A review_requested SSE event is broadcast to connected clients
  5. The ReviewGate UI component shows the JSON payload and action buttons

Submit actions: approved, rejected, edited (with a modified payload), or regenerate (ask the step to produce new output).

Reviewing individual clips

To review b-roll clip by clip, add a review gate to the b-roll step (review: true). At the gate you see the planned clips and can approve them, edit the selection, reject, or regenerate — the same approve/edit/reject/regenerate choices as any other gate, in the run view.

What "rejected" means

A rejected step outputs nothing. On the next run, the step's cache is invalid and it runs again from scratch (a new AI agent call, new Veo generation, etc.).
